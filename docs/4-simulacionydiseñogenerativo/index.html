Paso a Paso Introducción a la Simulación en Fusion 360

  1. Definimos el material.

  2. Lo segundo que hacemos es definir las restricciones. Para esto escogemos las partes ancladas, las que no se mueven. Estas quedarán con el simbolo de candado que vemos abajo.

  1. Definimos la carga. Para esto escogemos la parte que se lleva todo el peso. En este caso escogemos la parte que se ve indicada por la flecha. Ponemos carga 1.000 N para este caso.

  1. Hacemos un Pre Check para comprobar que la instalación cumpla con los requisitos.

  1. Finalmente, al tener todo correcto, lo resolvemos.

Esto es lo que debiesemos tener al terminar. Acá se ve la forma en la que la carga afecta a cada parte de la pieza.

Acá podemos ver como la pieza se curvaría con el peso, la forma en la que reacciona a la carga.

Paso a Paso Diseño Generativo en Fusion 360

  1. Preservamos la geometría. Para este caso escogemos las piezas que aparecen en la imagen de abajo de color verde.

  1. Lo segundo que hacemos es usar el comando Geometría Obstáculo. Para esto escogemos las piezas restantes, estas quedaran de color rojo.

  2. Para el tercer paso, ponemos las restricciones. Para esto escogemos las partes ancladas, las que no se mueven. Estas quedarán con el simbolo de candado que vemos abajo.

  1. Definimos las cargas. Para esto escogemos la parte interior de las piezas que tienen la flecha azul en la imagen anterior. En este caso ponemos carga -1.000 N.

  1. Definimos los objetivos y límites del objeto para optimizar los resultados. En este caso minimizamos la masa y usamos un coeficiente de seguridad 2.

  1. Especificamos las restricciones de fabricación. Para este caso queremos estimación de costes, sin restricciones, aditivo y fresado. Todo esto lo podemos observar en la imagen de abajo.

  1. Escogemos el material. En este caso escogemos 4, un titanio, iconel, plástico y aluminio.

  1. Hacemos un Pre Check para comprobar que la instalación cumpla con los requisitos.

  2. Por último, previsualizamos para generar una vista preliminar del resultado.
